1. Probar el proyecto ahora mismo (sin front-end)

A. Requisitos previos - Tener instalado Node.js (v16 o superior). -
Tener instalado MySQL y creado tu usuario y base de datos. - Crea la
base y la tabla: CREATE DATABASE mi_base; USE mi_base; SOURCE
schema.sql; – o pega el contenido de schema.sql

-   Edita el archivo db/connection.js con tus credenciales: const
    connection = mysql.createConnection({ host: ‘localhost’, user:
    ‘root’, // tu usuario password: ‘123456’, // tu contraseña database:
    ‘mi_base’ // tu base });

B. Instalar dependencias npm install

C. Ejecutar el servidor npm start Deberías ver: Conectado a la BD MySQL
Servidor corriendo en http://localhost:3000

D. Probar con Postman o cURL - Registrar usuario: curl -X POST
http://localhost:3000/auth/register
-H “Content-Type: application/json”
-d ‘{“username”:“juan”,“password”:“1234”}’

-   Iniciar sesión: curl -X POST http://localhost:3000/auth/login
    -H “Content-Type: application/json”
    -d ‘{“username”:“juan”,“password”:“1234”}’ Respuesta: { “mensaje”:
    “Login exitoso” }

-   Ver perfil protegido: curl -X GET http://localhost:3000/perfil
    –cookie “connect.sid=TU_COOKIE_DE_LA_SESION” (Postman maneja cookies
    automáticamente.)

-   Cerrar sesión: curl -X POST http://localhost:3000/auth/logout
